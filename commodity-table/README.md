Local vendor files
==================

This folder expects the vendor libraries to be available at the following relative paths (from the site root):

- ../vendor/handsontable/handsontable.full.min.js
- ../vendor/handsontable/handsontable.min.css
- ../vendor/hyperformula/hyperformula.full.min.js
- ../vendor/dompurify/purify.min.js

If you'd rather upload a fully self-contained folder, copy the files from `vendor/` into this folder and update the script/link tags accordingly.
